# Fast API - Introducción

Este proyecto tiene como objetivo aprender los conceptos básicos de **FastAPI** mediante la creación de una API RESTful sencilla.  
La API permite comprender cómo se comunica un cliente con un servidor utilizando el protocolo HTTP y cómo FastAPI genera documentación automática con Swagger y ReDoc.

## Objetivo específico

Utilizar **FastAPI** para crear una API RESTful básica que permita la comunicación entre el cliente y el servidor mediante solicitudes HTTP.

## Tecnologías utilizadas

- Python
- FastAPI
- Uvicorn
- Git
- GitHub
